4.변수(variable)
-용도:기본형 데이터 및 참조형 데이터를 저장하는 용도.
-문법:     변수명 = 값
	변수명=변수명=변수명=값
	변수,변수2 = 값, 값2 (*중요)  #tuple

-모든 변수는 참조형 변수이다. 즉, 변수에는 실제값이 저장된 위치값을 저장하고 있다.
따라서 위치값을 출력하기 위해서는 id(변수) 사용한다.
-type(변수) 이용하여 변수의 데이터 타입 확인 가능
-변수명은 단어와 단어사이에는 _ 이용하여 연결하는 것을 권장. (camel권장 안함)
-isinstance(변수,타입) 이용해서 타입 비교

7.연산자
  가. 산술연산자
  나. 대입연산자 심화
    - packing:변수와 값의 갯수가 일치하지 않을때 *사용 
	    남은요소 전체를 리스트에 담는다
  다. 비교연산자
   - 실행결과는 논리값(True/False)반환
   - 같냐?
     == 연산자 : 값만 비교
     is 연산자 :  주소값 비교
  라. 논리연산자 (and,or,not)
   - True/False값만 논리값으로 사용되지 않는다.
      False로 처리 : 0,"",None,[ ],( ),{ }
      이외의 데이터(값)은 모두 Treu로 처리된다
      논리값으로 변경방법 : bool(값)
    - 값 and 값 또는 값 or 값 --> 반환값은 True/False가 아니고 지정된 값중 하나가 반환.
  마. 비트 와이즈 (bitwise)
    - 7,|,^,<<,>>
  바. 멤버쉽 연산자
    - 집합형에서 지정된 값이 존재여부 확인 --> True/False 반환
